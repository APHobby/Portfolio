About

I work hands-on across CAD/modeling, functional prototyping, and FDM-focused design. My sweet spot is turning messy real-world constraints (measurements, tolerances, materials, assembly) into clean, testable designs — and iterating quickly until it fits and works.

Competencies

  • Parametric CAD: assemblies, interfaces, fasteners, mechanisms (locking/rotation), fit & clearance
  • DFM for FDM: print orientation strategy, support minimization, material choice, strength vs. weight
  • Tolerance work: press-fits, snap-fits, functional prototypes with real testing and iteration
  • Prototyping workflow: measurement-driven modeling → test prints → evaluation → refinement
  • Practical troubleshooting: printers (mechanics/electronics/slicer), hardware fixes, basic electronics integration
  • Software building: small tools, web UIs, visualization prototypes, automation helpers

Tools & stack

Shapr3D Blender Cura SuperSlicer FDM tuning JavaScript Python HTML/CSS VS Code Git Google Maps API Google Cloud

What I’m great at

  • Turning vague problems into concrete constraints and testable prototypes
  • Designing parts that are actually manufacturable on an FDM printer
  • Refining fit/function through fast iterations and clear documentation
  • Building simple, useful software tools that support the workflow
