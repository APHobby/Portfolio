Categories

Filename template (optional)

Use __ to auto-fill fields: 

slug__Title__tag1,tag2__2025.mp4

Examples: 

bayonet-nozzle__Bayonet Lock Nozzle__Shapr3D,FDM__2025.glb
waterproof-case__Waterproof Case__FDM,Fit__2024__01.jpg
motion-study__Motion Study__Blender__2025.mp4

If you add multiple files with the same slug, they’ll be grouped into one project as a media gallery.

Where to copy files

Create folders: 

assets/animation/
assets/cad/
assets/prints/
assets/code/

Then copy your files there. The importer writes paths like: ../assets/<category>/filename.ext
